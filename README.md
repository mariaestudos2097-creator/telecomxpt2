# 📊 Telecom X - Previsão de Churn com Machine Learning

Este projeto faz parte do desafio **Telecom X – Parte 2**, cujo objetivo é desenvolver modelos de **Machine Learning capazes de prever quais clientes têm maior probabilidade de cancelar seus serviços (Churn)**.

A análise utiliza dados previamente tratados no projeto **Telecom X – Parte 1**, onde foi realizado o processo de ETL e análise exploratória.

---

# 🎯 Objetivo do Projeto

A Telecom X enfrenta um alto índice de evasão de clientes.  
O objetivo deste projeto é construir modelos preditivos que permitam:

- Identificar clientes com maior risco de churn
- Antecipar cancelamentos
- Apoiar estratégias de retenção
- Reduzir perdas financeiras

---

# 🧠 O que é Churn

**Churn** representa o cancelamento de serviços por parte de clientes.
